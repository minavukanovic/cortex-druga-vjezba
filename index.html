HTML & CSS Basic Summary

HTML Summary

HTML (Hyper Markup Language) is used to define our page content ,structure and meaning.

You don't use it for styling purposes.Use CSS for that insted!

  • HTML uses "elements" to describe(annotate) content
  • HTML elements typically have an opening tag ,contentand then a closing tag
  • You can also have void(empty) elements like images
  • You can also configure elements with attributes
  • There's a long list of available elements but you'll gain experience over time,no worries.

Learn more about all available HTML elements on the MDN HTML elements reference

CSS Summary

CSS(Cascading Style Sheets) is used for styling your page content .

  • Styles are assigned via property-value pairs
  • You can assigned via the "style" attribute
  • To avoid code duplication ,you use global styles(e.g. via the "style" element) insted
  • Alternatively,you can work with external stylesheets files which you "link" to
  • When working with CSS,conteps like "inheritance",specifity and the "box model" shoud be understood.

Learn more about all available CSS properties and valueson the MDN CSS propery reference
